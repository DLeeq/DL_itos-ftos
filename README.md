# DL_itos-ftos
 Две функции:
- Для преобразования целых 32-битных чисел в строку Си
- Для преобразованяи чисел с плавающей точкой в строку Си

Точность преобразования чисел с плавающей точкой в диапазоне от 0 до 6 знаков после запятой.

Ничего лишнего:)
